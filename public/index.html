Gitpod Knowledge Base MCP Server

This is a Model Context Protocol (MCP) server that provides access to the Gitpod knowledge base.

Connection Information

To connect to this MCP server, use the following URL:

https://kumquat-vectorize-mcp.netlify.app/sse

Configuration for Claude Desktop

Add the following to your Claude Desktop configuration:

{
  "mcpServers": {
    "gitpod-kb": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://kumquat-vectorize-mcp.netlify.app/sse"
      ]
    }
  }
}

Testing

You can test the connection using the MCP inspector:

npx @modelcontextprotocol/inspector@latest

Then connect to https://kumquat-vectorize-mcp.netlify.app/sse
